CRUD.STORE - Fashion E-commerce Landing Page

CRUD.STORE es una interfaz de tienda de ropa moderna y minimalista. Este proyecto fue desarrollado enfocándose en el dominio de layouts complejos, diseño responsivo y la experiencia de usuario (UX) mediante el uso de CSS puro.
 Características Principales

    Diseño Responsive: Adaptabilidad completa para dispositivos móviles y escritorio utilizando Media Queries.

    Mosaico "Browse by Style": Implementación de un grid dinámico para categorías de estilo (Casual, Formal, Party, Gym).

    Carrusel de Testimonios: Un sistema de reseñas con scroll horizontal suave (scroll-snap) y efectos de desvanecimiento visual sin dependencia de JavaScript.

    Interactividad CSS: Efectos de hover en productos, botones redondeados y menús desplegables (submenús).

    Tipografía Moderna: Integración de Google Fonts (Lilita One y Ubuntu) para un look profesional.

 Tecnologías Utilizadas

    HTML5 Semántico: Estructura clara para mejor SEO y accesibilidad.

    CSS3 Custom Properties: Uso de variables y flexbox/grid para el diseño.

    Font Awesome: Iconografía para ratings, redes sociales y acciones de usuario.
